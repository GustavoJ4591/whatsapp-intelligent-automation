# Sistema de Automação Inteligente via WhatsApp

Projeto em desenvolvimento que utiliza lógica contextual, IA aplicada e integrações API para automatizar o fluxo de atendimento da empresa J.A., conectando WhatsApp Business API, banco de dados e ERP.

## Objetivo
Criar um sistema capaz de entender o contexto do cliente, organizar atendimentos, evitar duplicidades e integrar setores internos.

## Funcionalidades previstas
- Atendimento automatizado via WhatsApp
- Reconhecimento do cliente, região e histórico
- Fluxos dinâmicos (Orçamento, Manutenção, Locação, Outros Serviços)
- Coleta de informações (fotos, descrição, modelo, região)
- Roteamento por setor e vendedor
- Bloqueio de duplicidade de atendimentos
- Envio automático de e-mails
- Registro e análise de histórico
- Preparação para painel web e integração com ERP

## Tecnologias Planejadas
- Node.js
- JavaScript
- Python (IA aplicada)
- SQL
- Banco de Dados relacional
- WhatsApp Business API (Meta)
- REST APIs e Webhooks
- Servidor dedicado
